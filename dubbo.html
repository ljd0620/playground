Dubbo 是 Apache 顶级项目，高性能 Java RPC 框架。核心设计：面向接口代理、集群容错、负载均衡、SPI 可扩展、注册中心多适配。调用链：Consumer Proxy → Cluster → LoadBalance → Filter → Protocol → Registry 发现 → Provider。
🌐 Proxy Javassist 动态代理
🔊 Cluster 集群容错策略
⚖️ LoadBalance 负载均衡算法
🔧 Filter 链 监控/鉴权/限流
📦 Protocol 序列化 + 编解码
📡 Transport Netty 网络传输
