加载器层级

Bootstrap → JAVA_HOME/lib (rt.jar)
Extension → JAVA_HOME/lib/ext
Application → -classpath / CLASSPATH
Custom → 自定义路径 / 网络 / 动态生成

核心规则

① 收到加载请求，先委派给父加载器
② 父加载器递归向上委派直到 Bootstrap
③ 父加载器能加载则返回
④ 父加载器无法加载，才由当前加载器尝试
