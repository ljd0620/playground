Feign 是 Netflix 开源的声明式 HTTP 客户端，通过接口 + 注解即可定义 HTTP 调用，运行时由 JDK 动态代理生成实现。Spring Cloud 对其进行了增强，集成了负载均衡、服务发现和熔断降级。
@FeignClient(name = "user-service", url = "http://user-api")
public interface UserClient {
  @GetMapping("/users/{id}")
  UserDTO getUser(@PathVariable("id") Long id);
}
